Иногда есть необходимость опубликовать тот или иной контент на сайте на PHP в определённую дату, в определённое время. Чем заниматься этим вручную по напоминанию, можно поместить этот контент сейчас, но через `if` оператор проверять текущее время.

```php
<?php
/** Этот код в начало страницы (можно перед doctype html) */

$today = strtotime(date("Y-m-d")); // Текущая дата
$publishDate = strtotime("2017-11-01"); // Дата, когда нужно опубликовать этот контент. Переменную можно называть как угодно

/** Если дата релиза наступила, то новость можно выкладывать */
if ($publishDate <= $today): ?>

  <p>html-код здесь</p>

<?php else: ?>

  <!-- ничего не отобразится -->

<?php endif; ?>
```

Также через `date_default_timezone_set` можно задать конкретный часовой пояс, если его нужно учесть.
