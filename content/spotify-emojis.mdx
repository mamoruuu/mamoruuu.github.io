Точно работает в Spotify, установленный из их apt-репозитория. Нужно будет проверить snap- и -flatpak версию.

Это не баг Spotify.

1. Нужно установить шрифт с цветными эмодзи. Я установил _Noto Color Emoji_.

2. Нужно сообщить системе об эмодзи в шрифте. Для этого нужно создать файл `~/.config/fontconfig/fonts.conf` (файл в этом гисте)

3. Применить этот файл командой `fc-cache -f -v`.

После перезапуска приложения цветные эмодзи успешно отобразятся.
